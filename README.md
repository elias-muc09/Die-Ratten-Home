# Die Ratten — German Lesson Website

A small educational website for presenting the Projekt **_Die Ratten_** by **Gerhart Hauptmann**, for my German lessons.

## About the Project

This website presents selected text and source material from the Project Gutenberg edition of _Die Ratten_ by Gerhart Hauptmann.

## Source Text

The original text used on this site comes from Project Gutenberg’s edition of _Die Ratten_ by Gerhart Hauptmann [web:1][web:4].  
_Die Ratten_ is a Berlin tragicomedy first published in 1911 [web:1][web:4].

## About the Work

_Die Ratten_ is a naturalistic drama set in Berlin and is known for its social criticism and depiction of urban poverty [web:2][web:5].  
The title refers to rats both literally and symbolically, representing decay, hardship, and moral decline [web:2][web:5].
